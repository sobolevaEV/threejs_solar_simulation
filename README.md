# Three.js solar simulation.

This project is a small web application for simulating sunlight. 

### Technologies
- TypeScript
- Three.js
- Vite

### Functions
- Interactive camera control (OrbitControls)
- Simulation of a sun with Directional Light
- Realistic sun moition
- Adjustment of light color and intensity based on the time
- Shadow cast

### Why
I've created this project to make myself a little bit comfortable with three.js.

### How to start
```bash
npm install
npm run dev
